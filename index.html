即時掌握列車到站資訊

「到站未」為你提供最精準的香港港鐵實時到站資訊，讓你輕鬆規劃行程，準時抵達目的地。

功能特點

實時港鐵資訊

提供所有港鐵路綫的實時列車到站預測，包括月台資訊及列車狀態指示。

收藏常用車站

將常用車站加入收藏，一鍵查看最新到站時間，方便你的日常通勤。

快速查看功能

不加入收藏也能快速查詢任何車站的實時資訊，操作簡便直觀。

多語言支援

支援繁體中文、簡體中文和英文三種語言，滿足不同用戶的需求。

準確到站預測

顯示列車精確抵達時間和實際分鐘數，以及列車延誤狀態提醒。

即將新增巴士功能

未來版本將整合巴士實時到站資訊，打造全方位交通資訊平台。

關於我們

「到站未」是一款專為香港市民和旅客設計的交通資訊應用程式，致力於提供最準確、最及時的公共交通資訊。

我們的目標是透過科技創新，讓每位用戶能夠輕鬆掌握公共交通資訊，優化出行體驗。

應用程式所使用的資料均來自港鐵公司提供的公開資料，並會持續更新以保持資訊的準確性。

技術資訊

  • 資料來源： 港鐵實時列車服務資訊 API
  • 支援平台： Android 11.0+
  • 未來更新： 巴士、小巴資訊整合、路線規劃功能
